# GiftCard Management System

A full-stack solution for managing and tracking digital gift cards.

## Features
* **Card Issuance:** Create and track new gift cards.
* **Card History:** Preview past-used and expired cards.
* **Balance Tracking:** Real-time monitoring of balances and transaction history.
* **Secure Redemption:** API-driven validation for card usage.

## Local Setup Instructions

### 1. Backend Setup
Navigate to the `backend` folder and install the required Python packages:
``` bash
cd backend
pip install -r Requirments.txt
```
Run the application:
``` bash
python main.py
```

### 2. Frontend Setup
Navigate to the frontend directory and install the node modules:
```bash
cd frontend
npm install
```
Start the development server:  
``` bash
npm run dev
```

#### Project Structure
/backend: Contains the API logic and database models.

/frontend: Contains the React source code and assets.
